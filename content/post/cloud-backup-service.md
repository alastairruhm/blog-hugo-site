# 云备份服务比较

目前主流服务是 [CrashPlan](https://www.code42.com/crashplan/) 、[Backblaze](https://www.backblaze.com/) 和 [Arq](https://www.arqbackup.com/)

CrashPlan 和 Backblaze 功能基本相同，连收费策略都几乎相同。Arq 则可以自己定义备份的存储位置。
